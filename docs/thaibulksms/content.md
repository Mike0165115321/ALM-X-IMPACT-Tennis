# thaibulksms Documentation

## API Reference
- [Delete Attachment](https://developer.thaibulksms.com/reference/attachmentcontroller_delete.md): Delete a file attachment from the system
- [Get Attachment by UUID](https://developer.thaibulksms.com/reference/attachmentcontroller_get.md): Retrieve attachment information by UUID
- [List Attachments](https://developer.thaibulksms.com/reference/attachmentcontroller_list.md): Retrieve all file attachments uploaded to the system
- [Upload Attachment](https://developer.thaibulksms.com/reference/attachmentcontroller_upload.md): Upload a file attachment for use in sending emails
- [Check Credit Balance](https://developer.thaibulksms.com/reference/creditcontroller_getcredit-1.md): Check the remaining credit in the system  **Specify api-key (as username) and api-secret (as password), then click the "Try it" button**
- [Send Email](https://developer.thaibulksms.com/reference/transactionalcontroller_send-1.md): Send email using template  **Specify api-key (as username) and api-secret (as password), then click the "Try it" button**
- [Request OTP](https://developer.thaibulksms.com/reference/otpcontroller_send-1.md): Send a request to generate a PIN and send it to the recipient email  **Specify api-key (as username) and api-secret (as password), then click the "Try It" button**
- [Verify OTP](https://developer.thaibulksms.com/reference/otpcontroller_verify-1.md): Verify that the PIN received from the recipient email matches the PIN sent by ThaibulkSMS  **Specify api-key (as username) and api-secret (as password), then click the "Try It" button**
- [Error Reference Guide](https://developer.thaibulksms.com/reference/error-reference-guide-1.md): This document describes error codes, error messages, and response formats that may occur when using the TBS Email API.
- [ลบไฟล์แนบ](https://developer.thaibulksms.com/reference/attachmentcontroller_delete-1.md): ลบไฟล์แนบจากระบบ
- [ดึงไฟล์แนบตาม UUID](https://developer.thaibulksms.com/reference/attachmentcontroller_get-1.md): ดึงข้อมูลไฟล์แนบตาม UUID
- [ดึงรายการไฟล์แนบ](https://developer.thaibulksms.com/reference/attachmentcontroller_list-1.md): ดึงรายการไฟล์แนบทั้งหมดที่อัปโหลดไว้ในระบบ
- [อัปโหลดไฟล์แนบ](https://developer.thaibulksms.com/reference/attachmentcontroller_upload-1.md): อัปโหลดไฟล์แนบเพื่อใช้ในการส่งอีเมล
- [Verify Link (TH)](https://developer.thaibulksms.com/reference/copy-of-verify-link-th.md): การกดปุ่มหรือลิงค์การยืนยันใน email
- [Request OTP (TH)](https://developer.thaibulksms.com/reference/otpcontroller_send.md): ส่งคำขอเพื่อสร้าง PIN และส่งไปยังอีเมลปลายทาง  **ระบุ api-key (ใช้แทน username) และ api-secret (ใช้แทน password) จากนั้นให้กดที่ปุ่ม "Try It"**
- [Verify OTP (TH)](https://developer.thaibulksms.com/reference/otpcontroller_verify.md): ทำการยืนยันว่า PIN ที่ได้รับจากอีเมลปลายทางนั้นตรงกันกับ PIN ที่ทาง ThaibulkSMS ส่งให้  **ระบุ api-key (ใช้แทน username) และ api-secret (ใช้แทน password) จากนั้นให้กดที่ปุ่ม "Try It"**
- [เช็คเครดิตคงเหลือ](https://developer.thaibulksms.com/reference/creditcontroller_getcredit.md): ตรวจสอบเครดิตคงเหลือในระบบ  **ระบุ api-key (ใช้แทน username) และ api-secret (ใช้แทน password) จากนั้นให้กดที่ปุ่ม "Try it"**
- [ส่งอีเมล](https://developer.thaibulksms.com/reference/transactionalcontroller_send.md): ส่งอีเมลโดยใช้ template  **ระบุ api-key (ใช้แทน username) และ api-secret (ใช้แทน password) จากนั้นให้กดที่ปุ่ม "Try it"**
- [Error Reference Guide](https://developer.thaibulksms.com/reference/error-reference-guide.md): คำอธิบาย error codes, error messages และ response format ที่อาจเกิดขึ้นเมื่อใช้งาน TBS Email API
